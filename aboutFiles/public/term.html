Introdução:



Com esses comandos executados em uma interface de linha de comandos é possivel executar diretamente e independentemente alguns metodos das classes do pacote de manipuladores.
Para executar esses comandos caso esteja usando um executavel .jar, basta usar esse comando em seu terminal: "java -jar UmasFilasAi.jar { comandos }"
Caso esteja usando uma IDE, você devera configurar ela para que passe os comandos na hora da sua execução.
Todos os comandos são encadeados, ou seja, você pode passar varios e todos será exucutados em sequencia

Dica:
caso deseje guardar algum resultado em alguma variavel do seu shell, você pode executar o seguinte comando: lData=`java -jar UmasFilasAi.jar -o -k string1 string2`. Esse comando irá armazenar a chave gerada na variavel lData, Para usa-lá basta usar "$lData", como por exemplo para ver o que esta armazenada nela: "echo $lData".



Comando Função Exemplo
-e ou --execute Permite abrir a inteface grafica do programa após a exeucução do argumentos -
-o ou --omite troca entre Omitir a mensagem do resultado, mostrando assim apenas o resultado. Caso usado mais de uma vez vai trocando a visibilidade -
-k ou --keyGerate Gera uma chave de cifragem baseada em duas Strings -k string1 string2
-c ou --cifre Cifra uma String com base em um numero inteiro -c senha 42
-d ou --decifre Decifra uma String com base em um numero inteiro -d }yxy 10
-h ou --headerGet Mostra o cabeçalho de um arquivo. (Segundo parametro é opcional, ele sera usado como chave para arquivos não publicos) -h list.listdata
